PRODUCT CARD PROJECT:

A simple responsive product card built using HTML and CSS.
The card displays a product image, title, description, price, and action buttons (Add to Cart and Buy Now).

FEATURES:

1. Clean and modern card layout.
2. Responsive design (works on mobile &  Desktop).
3. Hover effects for interactive feel.
4. Two action buttons:
    i.  Add to Cart (left side)
    ii. Buy Now (right side)

PROJECT STRUCTURE:

project-card/
|-- index.html   # Main HTML
file

|--styles.css    # CSS
styling file

|--README.md     # Project
documentation

HOW TO RUN
1. Clone this repository:
    git clone
    https://github.com/Vidya933/product-card-project.git

2. Open the folder:
    cd product-card

3. Run the project:
    Open index.html in any browser (Chrome, Edge, Firefox).
